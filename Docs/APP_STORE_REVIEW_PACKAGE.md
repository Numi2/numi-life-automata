### Description

Numi Automata is a bottom-up artificial-life simulation for exploring how
organized life-like behavior can emerge from local physical rules.

Begin with coupled energetic, chemical, and environmental fields. A founder
cell appears only when local conditions physically support it. From there,
cells metabolize, move, signal, build membranes, form multicellular bodies,
crossbreed, reproduce, inherit developmental programs, and diverge across
different terrain.

Move continuously between ecological, organism, cellular, intracellular, and
wave scales. Accelerate the same simulation up to 24x, inspect live scientific
measurements, and follow lineages without assigning organisms an artificial
fitness score.

Numi Automata is an exploratory artificial-life system. Its quantities are
dimensionless simulation variables; it does not claim to model biological
abiogenesis or quantum computation.

### Review notes

Numi Automata is an offline, sandboxed Metal simulation. It has no account,
network service, analytics, advertising, payments, or user-generated content.

On launch, the main simulation opens automatically. Use the scale controls to
move between ecology, organism, cellular, intracellular, and wave views. Use
the speed control to run the unchanged solver at 1x, 3x, 6x, or 24x. The
scientific inspector is optional.

The world intentionally begins without initialized cells. A founder cell is
allocated when local field thresholds are met; this can take a short time.

The app requires Apple silicon and macOS 26 because its simulation and renderer
use current Metal features.
